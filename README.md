# Person Name Generator

A DevToys extension that generates random person names for testing purposes.

## Features

- Generate realistic random person names
- Powered by the Bogus library for data generation
- Fully integrated with DevToys

## Installation

Install this extension in DevToys to quickly generate random person names for your testing needs.

## Usage

Use this generator within DevToys to create fake names for test data, mocking, and other testing scenarios.

## Technologies

- .NET 8.0
- DevToys.Api
- Bogus
